## Recommended next steps to get `.site` ranking

1. **Confirm robots.txt allows Googlebot.** Upload the provided `robots.txt` (Allow: / + sitemap reference) to the domain root.
2. **Submit the sitemap** (`sitemap.xml`, provided separately) in Google Search Console under Sitemaps.
3. **Request indexing** for the homepage URL directly via Search Console's URL Inspection tool.
4. **Set the canonical tag** to `.site` (included in the meta tags file) so Google doesn't treat `.org` and `.site` as duplicate/competing content.
5. **Build a few backlinks** — the YouTube channel, Facebook page, and any partner/church-directory listings should link to `.site` directly.
6. **Re-check indexing status** in 1–2 weeks after submission; new domains typically take days to a few weeks for first crawl.

## Note on brand duplication

This church name has previously been targeted by scam/impersonation accounts (per an Africa Check report on a fake Facebook page using the same branding). Running both `.org` and `.site` as separate live domains for the same ministry can make it harder for visitors to tell which is authentic — worth confirming with the client which domain is the canonical one going forward, and considering a redirect from the other to avoid confusion.
